Guía de este proyecto:

  • Los 50 proyectos tienen como base 3 archivos: Uno HTML, uno CSS y uno JavaScript.
  • Selección de elementos del DOM: Utiliza `document.querySelectorAll('.screen')` para seleccionar todos los elementos con la clase 'screen', `document.querySelectorAll('.choose-famous-btn')` para seleccionar todos los botones con la clase 'choose-famous-btn', y `document.getElementById('start-btn')`, `document.getElementById('game-container')`, `document.getElementById('time')`, `document.getElementById('score')`, y `document.getElementById('message')` para seleccionar elementos específicos por su ID.
  • Manejo de eventos de clic: Añade un event listener al botón de inicio (`start_btn`) para que, al hacer clic en él, la primera pantalla se desplace hacia arriba, lo que permite al usuario avanzar al siguiente paso del juego.
  • Creación de elementos dinámicamente: Define la función `createFamous()` para crear elementos de celebridades (famosos) en posiciones aleatorias dentro del contenedor de juego. Cada celebridad se crea como un `
    ` con una imagen aleatoria y un estilo de rotación aleatorio. Estos elementos se añaden al contenedor de juego (`game_container`) y se les añade un event listener para detectar clics del usuario.
  • Actualización de la puntuación: Define la función `increaseScore()` para aumentar la puntuación del jugador cada vez que se atrapa a una celebridad. La puntuación se muestra en el elemento con el id "score". Si la puntuación supera 19, se muestra un mensaje de victoria al añadir la clase 'visible' al elemento con el id "message".

Espero que con esto te ayude a pensar cómo hacer este proyecto. Animate a equivocarte y sino, ya sabes que podés revisar el código del home!
