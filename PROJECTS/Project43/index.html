Guía de este proyecto:

  • Los 50 proyectos tienen como base 3 archivos: Uno HTML, uno CSS y uno JavaScript.
  • Selección de clasificación: Se utiliza un event listener en el contenedor de clasificaciones para capturar clics en los elementos de clasificación.
  • Actualización de selección: Cuando se hace clic en una clasificación, se actualiza la selección eliminando cualquier estado activo previo y marcando la nueva selección como activa.
  • Obtención de la calificación seleccionada: Se obtiene la calificación seleccionada del elemento de texto adyacente al elemento de clasificación activo.
  • Envío del feedback: Al hacer clic en el botón de enviar, se muestra un mensaje de agradecimiento junto con la calificación seleccionada por el usuario.

Espero que con esto te ayude a pensar cómo hacer este proyecto. Animate a equivocarte y sino, ya sabes que podés revisar el código del home!
