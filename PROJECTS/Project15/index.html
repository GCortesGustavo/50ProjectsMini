Guía de este proyecto:

  • Los 50 proyectos tienen como base 3 archivos: Uno HTML, uno CSS y uno JavaScript.
  • Iteración sobre los contadores: Utiliza `forEach` para iterar sobre todos los elementos con la clase "counter". Dentro del bucle, inicializa el texto de cada contador a "0".
  • Actualización del contador: Define una función `updateCounter()` para cada contador. Dentro de esta función, obtén el valor objetivo del contador desde el atributo `data-target` del contador. Luego, actualiza el contador incrementando su valor gradualmente hasta alcanzar el valor objetivo. Utiliza `setTimeout()` para llamar recursivamente a `updateCounter()` cada milisegundo para lograr la animación gradual.
  • Incremento gradual del contador: Calcula el incremento necesario para alcanzar el valor objetivo dividiendo el valor objetivo por 200. Esto asegura una animación suave y gradual del contador.
  • Finalización de la animación: Una vez que el contador alcanza el valor objetivo, establece su texto en el valor objetivo para asegurar que esté exactamente en el valor deseado.

Espero que con esto te ayude a pensar cómo hacer este proyecto. Animate a equivocarte y sino, ya sabes que podés revisar el código del home!
