Si no lográs hacerlo por tu cuenta, podés hacer dos cosas:

  • Descansar unos minutos
  • Entrar al repositorio de la página principal y leer el código

Se que no es fácil hacer un código sin tener una guía, yo no soy experto y tampoco un profesor 😁, pero puedo ayudarte a seguir una serie de pasos que pueden ayudarte a lograr el proyecto.

Guía de este proyecto:

  • Los 50 proyectos tienen como base 3 archivos: Uno HTML, uno CSS y uno JavaScript.
  • Crear y Seleccionar los elementos del encabezado, el título, el extracto, la imagen de perfil, el nombre y la fecha.
  • Seleccionar los elementos con clases "animated-bg" y "animated-bg-text".
  • Configurar un temporizador para llamar a la función getData después de 2500 milisegundos.
  • En la función getData, asignar contenido a los elementos seleccionados para simular la carga de datos.
  • Eliminar las clases "animated-bg" y "animated-bg-text" de los elementos para detener la animación de carga.

Espero que con esto te ayude a pensar cómo hacer este proyecto. Animate a equivocarte y sino, ya sabes que podés revisar el código del home!

¡Suerte!
