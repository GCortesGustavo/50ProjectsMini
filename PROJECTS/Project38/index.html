Se que no es fácil hacer un código sin tener una guía, yo no soy experto y tampoco un profesor 😁, pero puedo ayudarte a seguir una serie de pasos que pueden ayudarte a lograr el proyecto.

Guía de este proyecto:

  • Los 50 proyectos tienen como base 3 archivos: Uno HTML, uno CSS y uno JavaScript.
  • Asignación de un evento de clic a cada elemento de la lista de navegación
  • Al hacer clic en un elemento de la lista, se ocultan todos los contenidos y elementos de la lista
  • Se agrega la clase "active" al elemento de la lista clicado y la clase "show" al contenido correspondiente
  • Funciones auxiliares para ocultar todos los contenidos y elementos de la lista

Espero que con esto te ayude a pensar cómo hacer este proyecto. Animate a equivocarte y sino, ya sabes que podés revisar el código del home!

¡Suerte!
