Guía de este proyecto:

  • Los 50 proyectos tienen como base 3 archivos: Uno HTML, uno CSS y uno JavaScript.
  • Usa forEach para iterar sobre cada uno de los elementos label seleccionados. Esto te permitirá aplicar cambios a cada label individualmente.
  • Utiliza innerText.split("") para dividir el texto de cada label en caracteres individuales. Luego, usa map para crear un nuevo arreglo de caracteres, envolviendo cada letra en un elemento con un estilo transition-delay calculado en función de su índice.
  • Usa join("") para convertir el arreglo de elementos de nuevo en una cadena y actualiza innerHTML del label con esta cadena. Esto aplica la animación de retraso de transición a cada letra.

Espero que con esto te ayude a pensar cómo hacer este proyecto. Animate a equivocarte y sino, ya sabes que podés revisar el código del home!
