Guía de este proyecto:

  • Escucha los eventos de teclado: Utiliza `addEventListener()` para escuchar el evento `keyup` en el área de texto (`textarea`). Cuando el usuario presiona la tecla "Enter", elimina el contenido del área de texto y ejecuta la función `randomSelect()`.
  • Crea un elemento HTML para cada etiqueta: Dentro de la función `createTags(input)`, divide el valor del área de texto en etiquetas individuales separadas por comas (`,`) utilizando el método `split()`. Luego, filtra las etiquetas para eliminar cualquier espacio en blanco alrededor de cada etiqueta utilizando el método `trim()`. Después, crea un elemento `` para cada etiqueta y añádelo al elemento `
    ` con el id "tags" (`tagsEl`).
  • Selecciona aleatoriamente una etiqueta: Dentro de la función `randomSelect()`, utiliza `setInterval()` para seleccionar aleatoriamente una etiqueta cada 100 milisegundos durante un total de 30 veces. Para cada iteración, selecciona una etiqueta aleatoria utilizando la función `pickRandomTag()`, resalta temporalmente la etiqueta seleccionada utilizando la función `highlightTag()`, y luego desmarca la etiqueta después de un breve retraso utilizando la función `unHighlightTag()`.
  • Funciones auxiliares: Utiliza las funciones `pickRandomTag()`, `highlightTag(tag)` y `unHighlightTag(tag)` para facilitar la selección aleatoria y el resaltado de las etiquetas.
